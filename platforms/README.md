# Platform integrations

Ready-made wrappers for embedding the AccessibleWeb Widget on popular platforms, each loading the pinned jsDelivr build (`accessible-web-widget@1.3.1`) so you don't have to touch raw HTML. [`wordpress/`](wordpress/) contains a minimal WordPress plugin with a Settings > AccessibleWeb Widget page (position, button size, language) that enqueues the script and emits `window.AccessibleWebWidgetOptions` from your saved settings; [`shopify/`](shopify/) contains a theme snippet you paste into your theme and render before `</body>`, with optional `position`, `size`, and `lang` parameters. On any other platform, the plain `<script>` tag from the [main README](../README.md) works as-is.
